# 📱 Task Manager Flutter App

## 🚀 Overview

A Flutter-based Task Management application that allows users to create, edit, delete, search, and filter tasks with persistent local storage using Hive.

---

## 🛠️ Tech Stack

* Flutter (Dart)
* Provider (State Management)
* Hive (Local Database)

---

## 📌 Selected Track

**Track B: Mobile Specialist**

## ⚙️ Setup Instructions

1. Clone the repository:

```
git clone https://github.com/Manni-21/Flutter_task_manager_Application.git
```

2. Navigate to project:

```
cd Flutter_task_manager_Application
```

3. Install dependencies:

```
flutter pub get
```

4. Run the app:

```
flutter run
```

---

## ✨ Features

* Create, Update, Delete Tasks (CRUD)
* Search tasks by title
* Filter tasks by status
* Local persistence using Hive
* Loading state simulation
* Clean and responsive UI
