# 📱 Task Manager Flutter App

## 🚀 Overview

This is a Task Management mobile application built using Flutter.
The app allows users to create, update, delete, search, and filter tasks with a clean and responsive UI.

---

## 🛠️ Tech Stack

* **Frontend:** Flutter (Dart)
* **State Management:** Provider
* **Local Database:** Hive
